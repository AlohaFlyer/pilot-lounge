The Pilot LoungeAirport & Route Tools

Two quick map tools for the lounge. Search any airport by ICAO or IATA code, or plot a string of waypoints and airports into a route. Maps powered by Leaflet and OpenStreetMap tiles.

Airport Code Search

Type an ICAO 4-letter code (PHNL) or an IATA 3-letter code (HNL), then press Enter or tap Find. Case-insensitive.

Waypoint / Route Plotter

Enter space-separated points in order: named RNAV fixes (ALANA MALIA) or airport codes from the search above. Press Enter or tap Plot. The route line connects them in order.

Fix database is a curated Hawaii/US subset, not the full worldwide nav database. It is extensible: add more {id, lat, lng} entries to the FIXES list. Coordinates marked (approximate) are best-known Hawaii-area positions and should be verified against current charts before flight.
